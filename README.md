# Directory Structure

Follow this pattern for organization of developed components.

```
├── app
│   ├── css
│   ├── fonts
│   ├── img
│   ├── js
│   ├── libs
│   ├── scss
│   └──  index.html 
├──.editorconfig
├──.gitignore
├──.CHANGELOG.md
├──.gulpfile.js
├──README.md
├──package.json
└── 
```

